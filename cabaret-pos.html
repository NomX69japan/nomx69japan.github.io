🐙 GitHub 連携▼

POSファイルをGitHubリポジトリで管理します。変更履歴の保存・複数PC間でのファイル共有が可能です。

GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → repo スコープで生成

※ HTMLファイル全体をGitHubに保存します。データはHTMLに埋め込まれません（localStorageを使用）。
※ アップロードするとGitHub上のファイルが上書きされます。

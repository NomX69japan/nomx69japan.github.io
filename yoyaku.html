🔑 Google API 認証情報
設定手順：
1. Google Cloud Console を開く
2. 新規プロジェクト作成 → 「Google Calendar API」を有効化
3. 「認証情報」→「APIキー」を作成 → 下のAPI Keyに貼る
4. 「OAuth 2.0クライアントID」を作成（種類: ウェブアプリ）
   承認済みJSオリジン: file:// または公開先URL
5. クライアントIDを下に貼る → 「Googleでログイン」ボタンを押す
